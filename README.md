# PDF Diff Highlighter

Flask-приложение для сравнения двух PDF-чертежей.

## Возможности

- сравнение PDF по страницам;
- красная маска визуальных изменений;
- настройка точности;
- режим сравнения выбранной области с автопоиском соответствующей зоны;
- ручная корректировка найденной области;
- навигация по листам многостраничного PDF;
- полноэкранный просмотр и скачивание результата;
- временная обработка PDF без постоянного хранения загруженных файлов.

## Основной стек

Проект поддерживает одну реализацию: **Python 3.11 + Flask**.

## Запуск

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Локальный адрес: `http://127.0.0.1:8000`.

## Проверка

```bash
.venv/bin/python -m pytest -q
node --check static/app.js
git diff --check
```
