# PDF Diff Highlighter — C# версия

Отдельная версия в подпапке `csharp/`.

## Что внутри
- ASP.NET Core минимальное приложение
- сравнение текста по страницам
- таблица `Исходный / Измененный`
- показ только строк `Добавлено / Удалено`
- diff-изображение с полноэкранным просмотром и скачиванием

## Зависимости
- .NET 8 SDK
- NuGet: `UglyToad.PdfPig`, `PdfiumViewer`, `SkiaSharp`

## Запуск

```bash
cd csharp
dotnet restore
dotnet run
```

> На этой машине `dotnet` не установлен, поэтому сборку здесь не проверял.
